# 知识库检索工具 (search_knowledge_base)

## 功能
从团队的 Notion、SVN 或 Google Drive 知识库中检索文档、策划案或技术规范。

## 数据源
| 来源 | 说明 |
|------|------|
| notion | Notion 文档数据库 |
| svn | SVN 代码仓库和提交日志 |
| gdrive | Google Drive 云端文档 |
| all | 同时检索所有来源 |

## 执行规则
1. 根据用户问题提取核心关键词
2. 多源并行检索，返回相关片段
3. 读操作，不需要用户确认
4. 返回结果附带溯源信息 (Citations)

## 审计规则
- 仅允许搜索查询
- 不允许修改任何知识库数据
- 不允许通过知识库执行代码
