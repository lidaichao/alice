**覆盖的主要分类：**
- **TsubasaGame（大多数 UI 模块）**：支付/商城、登录后流程 UI、主界面/日常/任务、养成系统（球员、装备、训练等）、抽卡/招募、竞技/俱乐部/天梯、战斗入口 UI、活动/礼包、好友/聊天/邮件、通用 UI 组件等。
- **TsubasaBattle**：战斗数值、规则阶段、技能结算、球员属性与动作逻辑等异常。
