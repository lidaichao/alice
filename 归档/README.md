主工程仅保留 `backend/`、`frontend/`、`desktop/`、`eval/`、`scripts/`、`test/`、`docs/`。

| 子目录 | 内容 |
|--------|------|
| `archive/` | 原根目录 `archive/`：Java Jira 插件、Gemini 早期全栈快照、企微桥接实验 |
| `research/` | Claw 压测/诊断脚本、WorkBuddy 记忆；`baize`/`LobeHub` 为本地对照（gitignore） |
| `workspace/` | WorkBuddy 自动化与历史架构草稿 |
| `logs/` | 根目录遗留运行日志 |
| `backend-dev-scratch/` | 一次性联调脚本（含硬编码 Key 的勿提交副本） |

查阅 Java 插件：`归档/archive/jira-workbuddy-plugin/`。
